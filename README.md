# dev.finances
Projeto front-end web simples, desenvolvido em uma das maratonas da Rocketseat.</n>
Trata-se de uma calculadora de finanças pessoais desenvolvida em Javascript puro com localstorage para fins de prática front-end.</n>
Possui responsividade e recurso de acessibilidade para leitores de tela integrados aos navegadores.
